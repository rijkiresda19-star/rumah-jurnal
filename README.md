# Jurnal MEA

Jurnal MEA (JURNAL MANAJEMEN, EKONOMI, AKUNTANSI) — situs untuk berbagi artikel dan penelitian di bidang manajemen, ekonomi, dan akuntansi.

Live: https://rumah-jurnal-drab.vercel.app/

Pengembangan
- Edit `index.html`, `styles.css`, `script.js`
- Commit & push ke `main` → Vercel auto-deploy

Notes
- Contact form saat ini menggunakan client-side validation; replace `action` di form dengan Formspree ID atau setup serverless untuk menerima submission.
